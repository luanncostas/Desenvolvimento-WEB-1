Exemplo de Pseudo-elementos em CSS

"A educação é a arma mais poderosa que você pode usar para mudar o mundo."

Este é um exemplo de como o pseudo-elemento ::first-line pode ser utilizado para estilizar apenas a primeira linha de um parágrafo. O resto do texto permanece com o estilo padrão aplicado ao parágrafo.

A primeira letra deste parágrafo é estilizada de maneira diferente utilizando o pseudo-elemento ::first-letter.
